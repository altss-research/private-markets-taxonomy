# Private Markets Taxonomy

This repository provides a neutral, structured reference taxonomy for private-market allocators, fund structures, and asset classes.

It is designed for use by:
- Researchers
- Allocators
- Fund managers
- Data and analytics teams
- Documentation and schema designers

The taxonomy reflects commonly observed classifications across institutional private-market disclosures and public documentation.

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18178858.svg)](https://doi.org/10.5281/zenodo.18178858)

## Cite this

## Scope

This repository covers:
- Allocator types (family offices, pensions, endowments, sovereigns)
- Fund structures (closed-end, evergreen, continuation vehicles)
- Private-market asset classes
- High-level regional allocation frameworks
